# Trinidad & Tobago Fuel Cost Calculator

A sleek, responsive dark-themed webpage to quickly calculate estimated trip costs based on local fuel prices.

## Features
*   **Local Fuel Options**
*   **Clean UI**
*   **User Friendly**

## How to Run
Simply download the `index.html` file and double-click it to open it in any web browser.
